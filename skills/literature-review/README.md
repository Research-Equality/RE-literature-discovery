# literature-review

Theme-level synthesis over an already ranked corpus.

Recommended input is the canonical reranked `paper_db.jsonl` with `selection_bucket`, `authority_score`, `evidence_score`, and caution flags. Use `core` papers to anchor themes, `supporting` papers to widen coverage, and `background` papers for historical context.

This skill now also owns the repository's lighter analysis modes:
- comparison tables
- consensus / contested claims
- contradiction notes
- claim tracking
- gap analysis

See `references/analysis-modes.md` for the merged patterns.
