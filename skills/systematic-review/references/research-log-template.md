# Research Log Template

Chronological record of major review decisions and actions. Keep this file append-only.

Use it when a deep research topic spans multiple sessions, pivots, or substantial corpus revisions.

## Template

| # | Date | Type | Summary |
|---|------|------|---------|
| | | | |

Suggested entry types:

- `bootstrap`: initial scoping, search expansion, corpus bootstrapping
- `frontier`: recent-paper scan and frontier observations
- `survey`: corpus merge, deduplication, thematic clustering
- `deep-dive`: full-paper reading and representative-paper selection
- `synthesis`: taxonomy update, contradiction analysis, gap consolidation
- `pivot`: topic narrowing, question reformulation, or source-policy change
- `report`: major artifact completed for downstream writing

## Good Entry Pattern

Each entry should capture:

- what changed
- why it changed
- what evidence triggered the change
- what the next action should be

## Example

| 1 | 2026-03-27 | bootstrap | Searched Semantic Scholar, arXiv, and OpenAlex for long-context agent literature. Built an initial 52-paper corpus. Found that benchmark-heavy papers dominate, but theory-oriented memory-planning work is underrepresented. Next: expand targeted searches around memory architectures. |
| 2 | 2026-03-27 | deep-dive | Read 9 representative papers in full. Clear split emerged between retrieval-augmented, compression-based, and planning-centric approaches. Added this taxonomy to `phase5_synthesis/synthesis.md`. |
| 3 | 2026-03-28 | pivot | Narrowed topic from "long-context agents" to "memory architectures for long-horizon agents" because generic prompting papers diluted the corpus and weakened synthesis quality. |
